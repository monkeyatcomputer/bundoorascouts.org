# Security Policy

## Supported Versions

We currently only support the live version of the website.

| Version | Supported          |
| ------- | ------------------ |
| Live Site | :white_check_mark: |
| Older Versions | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities to **info@bundoorascouts.org**.

We will acknowledge receipt of your report within 5 working days and keep you updated on the progress of a fix.
